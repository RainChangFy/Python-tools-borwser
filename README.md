# Python-tools-browser

#### 说明

使用现在最新的版本做的

**Python**：python 3.8.2

**Pyqt5**：pyqt 5.14.2

使用的是QtWebEngineWidgets引擎，新版本的pyqt不再使用之前的版本了

#### 功能

制作了一个简单的浏览器，能够多标签的打开窗口，输入地址进入指定的网址，可以继续的浏览，实现了简单的前进，后退，停止，刷新的功能
